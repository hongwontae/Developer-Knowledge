1. NodeJS란?
=> 서버 사이드에서 JavaScript를 실행할 수 있게 해주는 런타임 환경입니다
=> JavaScript를 웹 브라우저가 아닌 서버에서도 사용할 수 있게 해줍니다.


2. 주요 특징
2-1. Node.js는 Chrome V8 JavaScript 엔진을 기반
2-2. 비동기 I/O:  파일 시스템, 네트워크 요청 등의 작업을 비동기적으로 처리하여 높은 성능과 효율성을 제공한다.
2-3. 이벤트 기반 아키텍처 : 이벤트와 콜백 함수를 통해 비동기 작업의 완료를 처리합니다.
2-4. 단일 스레드


3. 주요 구성 요소
3-1. V8 JavaScript 엔진
3-2. 이벤트 루프 : NodeJS의 비동기 작업의 매커니즘, 이벤트 루프는 계속해서 이벤트를 기다리고, 해당 이벤트가 발생하면 관련된 콜백 실행
3-3. 모듈 시스템
3-4. 내장 라이브러리
