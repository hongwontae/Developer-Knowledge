API (Application Programming Interface)
=> 사전적 정의 :  한 프로그램에서 다른 프로그램으로 데이터 주고받기 위한 방법을 말한다.

다른 프로그램에게 주기위해 설계된 것이라면 API라고 부를 수 있다.
프로그램들이 서로 소통하는 방법을 의미하기도 한다.

웹 개발자가 음식점 브라우저를 만들고 싶다. 이 떄 분점들을 표기하기 위해서 위치를 구현해야 한다. 이 떄 개발자가 직접 위치를 구현할 코드를 모조리 작성해야 할까? 위치를 구현하는 코드를 미리 구현한 회사나 개인에게 빌려쓸 수 있다. 이 코드를 사용해서 음식점 브라우저의 분점 표시를 완성하는 것이 바로 API를 사용했다는 것이다.

유용한 작업을 미리 구현한 것을 나의 프로그램 상에서 사용한다면 그것이 바로 API이다.
