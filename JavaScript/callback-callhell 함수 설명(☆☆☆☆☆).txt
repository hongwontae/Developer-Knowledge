1.콜백함수
=> 다른 함수에 인자로 전달되는 함수를 의미합니다.
=> 주로 비동기 함수 작업을 처리할 떄 콜백함수를 사용합니다.


2. 콜백 함수 목적
2-1. 동작 완료 후의 콜백 함수 실행
=> 후속 작업을 처리할 수 있다.
2-2. 유연성
=> 인자로 함수를 받아서 원하는 대로 처리할 수 있다.


3. 콜백 지옥
=> 콜백 함수를 중첩해서 사용하는 경우 가독성이 떨어지고 코드가 복잡해지는 경우를 의미합니다.
=> 콜백 지옥을 피하기 위해서 promise, async-await를 사용합니다.
