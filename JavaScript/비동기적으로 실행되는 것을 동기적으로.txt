1. 비동기적으로 실행되는 것을 동기적으로 코딩하는 방법이 있나요?
=> async/await를 사용하면 비동기 코드를 동기적으로 실행할 수 있습니다.
=> async가 붙은 함수 내에서 await를 사용하면 해당 함수의 호출 후 완료될 떄까지 다음 코드로 넘어가지 않고 기다립니다.



면접 질문
비동기적인 코드가 동기적으로 코딩하는 방법
=> 비동기 함수를 async await 방법으로 실행시키면 동기적인 코드처럼 사용할 수 있습니다.
=> async 함수 내에서 비동기 코드를 호출하기 전에 await라는 키워드를 붙이면 함수 실행은 네트워크 API에게 전달되고
함수의 실행이 일시정지됩니다. 그리고 await 이후의 코드들은 태스크 큐에 담기게 되고 콜 스택이 비워져 있을 떄 이벤트
루프를 통해 콜 스택에 올려지게 됩니다.
