1. JS 데이터 형변환
=> JS에서 두 가지 데이터 형 변환환 방식이 존재합니다.

2. 명시적 형변환 (Explicit Type Conversion): 개발자가 의도적으로 데이터 타입을 변환하는 것.
2-1. Number(), String(), Boolean()를 통해 이루어집니다.


3. 암시적 형변환 (Implicit Type Conversion): JavaScript 엔진이 자동으로 데이터 타입을 변환하는 것.
3-1. 주로 연산자 사용시 발생합니다. 
3-2. 예를 들어 +를 사용해서 숫자와 문자열을 결합하여 문자열을 만들거나 -를 사용해서 숫자와 문자열의 계산을 할 수 있
습니다.

JS 데이터 형변환에 대한 설명
=> JS에서 일어나는 형변환은 두 가지입니다. 하나는 명시적 형변환으로 Number, String, Boolean 등을 호출해서
형변환을 일으킵니다. 
=> 또 다른 하나는 암시적 형변환입니다. 하나는 JS 엔진에 의해 자동으로 형변환되는 것을 의미합니다.
=> 주로 연산자를 이용해서 두 타입을 비교하거나 계산할 떄 일어납니다.
