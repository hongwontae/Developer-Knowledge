1. 자바스크립트를 멀티 쓰레드처럼 사용하는 방법이 뭔가요?

1-1. web-Worker
=> 웹 워커를 사용하면 백그라운드에서 JS 코드를 실행할 수 있습니다.
=> 이를 통해 메인 쓰레드와 독립적으로 작업을 처리하고, UI의 응답성을 유지할 수 있습니다.
=> 웹 워커는 별도의 파일로 작성되며, Worker 객체를 사용하여 생성하고 postMessage를 통해 메인 쓰레드와 통신합니다.


1-2. 비동기 프로그램밍
=> Promise, async/awiat, browser API등을 사용해서 비동기적 작업을 처리하여 멀티 쓰레드처럼 작업 처리할 수 있습니다.
