1. 콜 스택
=> 함수 호출과 실행을 관리하는 데이터 구조입니다.
=> 함수 호출이 있을 떄마다 호출된 함수의 정보를 스택 형태로 저장합니다.
=> 콜 스택은 후입선출 방식으로 동작합니다. 가장 나중에 추가된 함수가 가장 먼제 제거됩니다.


2. 힙
=> 동적 메모리 할당을 위한 메모리 공간입니다.
=> 객체나 배열 같은 데이터를 저장합니다.


면접 질문
콜 스택과 힙 메모리에 대해서 설명하세요
=> 콜 스택은 함수 호출과 함수 실행을 관리하는 데이터 구조를 의미합니다. 가장 늦게 실행되는
함수가 가장 먼저 제거되는 후입선출의 흐름을 따릅니다. 콜 스택에서 함수가 호출되면 지역 변수, 매개 변수,
실행 컨텍스트 정보가 스택 형태로 저장되고 이를 스택 프레임이라고 합니다. 스택 프레임과 함께 함수가 실행됩니다.

=> 힙은 동적 데이터를 저장하기 위한 메모리 공간을 의미합니다. 배열이나 객체같은 데이터를 저장할 수 있습니다.
콜 스택에는 객체와 배열에 접근하기 위해 힙의 주소값이 저장되어있습니다.
