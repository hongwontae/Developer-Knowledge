1. 클래스
=> JS에서 객체 지향 프로그래밍을 지원하는 문법으로 객체를 정의하기 위한 청사진을 제공합니다.


2. 특징
2-1. 클래스 선언 : class className {}
2-2. constructor(){} : 생성자라고 부르며, 인스턴스가 생성될 떄 호출됩니다.
2-3. 메서드 : 클래스 내부에 선언된 함수로, 인스턴스에서 호출할 수 있습니다.
2-4. 인스턴스 : class의 청사진을 바탕으로 생성된 객체를 의미합니다. new 키워드와 className을 호출해서 생성합니다.
2-5. 상속 : 다른 클래스로부터 속성과 메서드를 물려받을 수 있습니다.
