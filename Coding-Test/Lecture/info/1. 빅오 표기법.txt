1. 빅오란?
1-1. 프로그램이 처리하는 연산 횟수를 산정한다.
1-2. 입력 크기 n이 커질수록, 실행시간이나 사용하는 메모리가 어떻게 증가하는지 표현합니다.

1-3. 자주 나오는 표기
1. O(1) : 입력과 상관없이 일정 시간
2. O(N) : 입력 크기에 비례합니다.
3. O(N 2제곱) : 이중 FOR 루프
4. O(LOG N) : 로그 시간
5. O(N LOG N) : 병합 정렬...

1-4. 로그개념
=> O(LOG N) -> n이 절반씩 줄어들 떄 발생합니다.
=> O(LOG N LOG) -> log n 단계 + 각 단계에서 O(n) 작업


2. 시간 복잡도
=> 알고리즘이 실행되기까지 걸리는 시간의 증가량


3. 공간 복잡도
=> 알고리즘이 사용하는 메모리 공간의 증가량


4. performance.now()
=> 현재 시간에서 정밀한 시간 경과 시간을 측정할 떄 사용하는 메서드
=> 마이크로 단위입니다.

=> let a =  performance.now()
