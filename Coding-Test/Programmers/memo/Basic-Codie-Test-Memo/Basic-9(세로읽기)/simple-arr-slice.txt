1. 문제
=> arr, 정수 n을 받아서 arr에서 n 만큼 요소를 return 하는 문제
=> slice에 대해서 알고 있어야 합니다.


2. 개념
=> slice는 arr 요소를 변형하여 새로운 배열을 반환하는 메서드입니다.
=> slice(start, end) start는 포함 인덱스, end는 비포함 인덱스입니다.
=> slice(0, 4) -> 0인덱스부터 3인덱스까지 가져와서 새로운 배열을 만들고 return
=> slice(0, -2) -> 0인덱스부터 arr.length-2를 의미합니다.


3. 나의 풀이
=> slice를 통해 간단히 반환
=> 다른 사람들의 풀이도 일맥상통
