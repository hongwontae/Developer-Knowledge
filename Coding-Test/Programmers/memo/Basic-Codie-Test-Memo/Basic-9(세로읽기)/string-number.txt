1. 문제
=> a,b 정수형 매개변수를 받고 ab, ba로 이어붙인다.
=> 그리고 a가 크면 ab, b가 크면 ba 동일하면 ab를 반환한다.


2. 나의 풀이
=> a,b 모두 String() 함수를 사용해서 형변환을 한 후 + 연산을 했다.
=> 문자열을 +하면 결합되는 속성을 이용
=> 그리고 if문 분기를 통해 값을 return, 이 떄 Number로 형변환


3. 다른 사람 풀이
=> Math.max를 사용 and `${}${}`를 사용한 것이 나와 달랐다. (Math.max(a,b) 두 값이 동일하면 일치하는거니까 그냥 반환)
=> 템플릿리터럴을 사용해서 자동으로 문자를 이어붙였고 Math.max를 통해 큰 수를 넘겨버렸다.
