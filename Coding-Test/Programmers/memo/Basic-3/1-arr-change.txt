1. 처음 풀이
=> 정수 배열 a를 받고 이차원 정수 배열 b ([[0,1], [0,4], ..])를 받아서 b의 요소를 인덱스로 하여 arr를 change하는 문제
=> 무식하게 품

1-1. 2차원 배열의 요소가 2개만 존재한다.
1-2. 0,1 그래서 map(ele=>ele[0], ele[1]) -> a = arr[ele[0]],  b = arr[ele[1]] 값을 찾고 변수에 할당
1-3. 기존 배열은 그대로이기 때문에 그냥 chage arr[ele[0]] = a; arr[ele[1]] = b;
=> 성공


2. 다른 사람 풀이
※ 구조 분해 할당에서 arr = [0,1,2,3,4] let a = 1; let b =3;
=> [arr[b], arr[a]] = [a,b] 이런 구문이 존재한다면
=> arr[b]는 a값이, arr[a]는 b값이 되고 자신의 위치를 지킵니다.
=> arr[b] = a; arr[a] = b와 같은 말입니다.
