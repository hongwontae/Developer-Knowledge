1. 구분자를 사용하여 문자열을 분리하고 사전순으로 정렬한 배열을 반환하는 방법


2. 개념
=> 배열을 정렬하는 sort에 대해서 알고 있어야 한다.
=> 또한 split('구분자')로 나눴을 떄 빈 문자열이 나올 수 도 있음을 알아야한다.

2-1. split('x') 빈 문자열 원리
1. 구분자가 연속적으로 나올 떄 그 사이에 빈 문자열이 생깁니다.
2. 구분자가 맨 앞에 있거나 맨 뒤에 존재하면 빈 문자열이 해당 장소에 생깁니다.

2-2. arr.sort()
=> 기본적으로 문자열 기준(유니코드)로 정렬합니다.
=> 대문자가 소문자보다 유니코드 값이 작습니다. (대-소문자 동일하게 여기지 않음)

=> sort 숫자 비교
=> 이 떄는 sort 내부에 콜백 함수를 사용해야 합니다.
=> sort((a,b) => a-b)
=> 숫자 오름차순
=> sort((a,b)=> b-a)
=> 숫자 내림차순


3. 나의 풀이
=> 받은 매개변수 문자열을 split()를 통해 분리하고 filter를 통해 falsy 값을 제거합니다.
=> 그리고 sort를 통해 유니코드 순서대로 정렬합니다.
=> 여기서 만약 대-소문자가 섞여 있다면 아래 작업을 해야 합니다.
=> items.sort((a, b) => a.toLowerCase().localeCompare(b.toLowerCase()));
