1. 첫 번쨰 문제
=> 대문자로 치환하는 문제
=> toUpperCase()라는 함수를 아는지 물어보는 것입니다.

2. 개념
=> toUpperCase(), toLowerCase()는 대-소문자로 치환하는 함수입니다.


3. 두 번쨰 문제
=> 배열을 매개변수로 주고 첫 번쨰로 나오는 음수의 인덱스를 반환하는 함수입니다.
=> findIndex를 사용할 줄 아는지 물어보는 것입니다.

4. 개념
=> findIndex는 return 값이 true라면 해당 index를 반환하거 아무것도 일치하지 않으면 -1를 반환하는 함수입니다.


5. 세 번쨰 문제
=> n, m 정수를 매개변수로 받고 n~m 까지 숫자를 배열에 반환하는 문제입니다.
=> slice(n,m)
