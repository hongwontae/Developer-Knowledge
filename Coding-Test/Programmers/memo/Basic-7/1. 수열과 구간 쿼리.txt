4. 문제
=> arr 정수 배열, queries 2차원 정수 배열이 주어집니다.
=> queries -> [s,e] 꼴이고 s와 e 사이의 숫자(포함)를 i라고 가정합시다.
=> 이 떄 arr[i]에 +1를 하세요


5. 나의 풀이
=> 일단 문제 해석 자체가 심하게 안됐다.
=> 그래서 s와 e 두 가지 값만 가지고 +1만 해버렸다. 당연히 해결안됨


6. 다른 사람 풀이
=> for문을 두 번 사용해야 한다.
=> map을 통해 s,e을 얻어오고 s가 작은 값, e가 큰값이기에 for에서 초기값은 s, 범위는 e를 넘아갈 떄까지
 반복하여 arr[i]+=1를 하고 return 한다.
