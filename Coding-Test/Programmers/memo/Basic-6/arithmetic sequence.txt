1. 등차수열이란?
=> 숫자들이 일정한 간격(같은 수)을 더해지면서 나열된 수열을 말합니다.
=> 2, 4, 6, 8, 10 / 공차는 -> 2
=> 공차는 더해지는 숫자를 의미합니다. 


2. 지식
=> new Array()를 통해서 배열을 생성하면 null, undefined도 아닌 비어 있는 상태가 됩니다.
=> 그래서 map, forEach, filter 같은 메서드들이 제대로 동작하지 않습니다.
=> 그래서 fill()로 값을 명시적으로 채워주는것이 좋습니다.


3. 나의 풀이
3-1. included.length을 통해 new Array()로 같은 길이를 가진 배열을 생성합니다.
3-2. 3-1 배열을 통해 map으로 초기값 + 등차로 등차수열을 생성합니다.
3-3. 그리고 등차수열 배열 filter을 사용해서 included의 bol 값과 일치를 확인하여 true 값만 모읍니다.
3-4. reduce를 통해 총합을 만들어내서 return 합니다.


4. 다른 사람 풀이
=> included 배열은 boolean 배열입니다.
=> included.reduce를 통해서 누적값과 현재값을 알 수 있습니다. 만약 현재값이 true라면 acc+a+b*idx를 통해 등차 수열
을 누적하여 계산할 수 있습니다.
=> 더 쉬운 풀이입니다. included가 boolean인 걸 알아차리고 idx를 이용할 줄 알아야 합니다.
