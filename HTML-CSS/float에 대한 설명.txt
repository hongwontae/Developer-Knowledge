1. float
=> 요소가 자신을 포함하고 있는 컨테이너 왼쪽이나 오른쪽에 배치되게 하는 속성입니다.
=> 이 떄 문서의 흐름에서 벗어나지만 필요한 공간은 차지합니다.


2. clear
=> clear 속성을 통해 이 후의 요소들이 문서 대열을 갖추게 만들 수 있습니다.
=> clear both를 사용하면 float이 적용된 요소들이 문서 대열을 갖췄다고 판단하여 밑으로 내려갑니다.
