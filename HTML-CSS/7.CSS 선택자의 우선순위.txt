1. CSS 선택자 우선순위
=> 여러 개의 CSS 규칙이 존재할 떄 어떤 스타일이 우선적으로 적용될지 결정하는 개념을 의미한다.


2. 순위
1. 인라인 스타일 (1000)
2. id 선택자 (100)
3. 클래스, 속성, 의사 클래스 선택자 (10)
4. 태그 선택자, 의사 요소 선택자 (1)
5. 전체 선택자, 연결된 결합자 (0)


3. !important
=> 해당 키워드를 사용하면 어떤 우선 순위보다 더 높은 우선순위를 갖는다.
=> 그러나 !important가 여러 개이고 충돌하면 다시 선택자 우선순위 규칙이 적용된다.
