1. box 모델
=> HTML 요소가 페이지에서 공간을 차지하는 방식을 설명하는 개념입니다.
=> 4개의 영역으로 구성된 사각형 박스로 표현되고, 각각 컨텐츠, 패딩, 테두리, 마진으로 나눌 수 있습니다.


2. 4개 영역
1. 컨텐츠 : 실제 내용
2. 패딩 : 컨텐츠와 테두리 사이의 공간
3. 테두리 : 패딩과 마진 사이에 위치한 선
4. 마진 : 요소와 다른 요소 사이의 공간


3. box-sizing
3-1. content-box : 기본값, width과 height은 컨텐츠 크기만 포함
3-2. border-box : width과 height이 패딩과 테두리까지 포함
=> 일반적으로 border-box를 사용하는 것이 좋다.
