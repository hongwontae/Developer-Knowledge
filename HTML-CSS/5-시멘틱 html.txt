1. 시멘틱 HTML
=> 웹 페이지의 구조를 의미적으로 잘 표현할 수 있도록 html 요소를 사용하는 방법


2. 주요 시멘틱 HTML 요소
2-1. header(로고, 제목, 탐색 메뉴)
2-2. nav(내비게이션 링크)
2-3. main(페이지의 주요 컨텐츠)
2-4. section(문서 내의 구획을 정의)
2-5. article(독립적으로 배포 가능한 컨텐츠, 뉴스기사)
2-6. footer (문서의 푸터)
2-7. figure(삽화, 이미지, 다이어그램)
2-8 Time Tage : <Time/>
=> 날짜나 시간을 나타낼 떄 사용하는 시멘틱 태그이다.
(datetime props
1. 기계가 읽을 수 있는 형식으로 날짜와 시간을 나타낸다.
2. ISO 8601를 표준을 사용
3. 검색 엔진은 Time tag의 datetime을 통해 시간 관련 콘텐츠를 이해한다.)


3. time 태그의 내부 텍스트
=> 사용자가 읽는 시간 정보이다.


3. 시멘틱 HTML의 장점
3-1. SEO 개선
(검색 엔진 크롤러가 시멘트 HTML를 보고 각 컨텐츠가 무슨 의미를 가지는지 이해할 수 있고 페이지에 대한 효율적이고 정확한 평가가
가능하다.)
3-2. 유지보수 용이
3-3. 코드의 가독성 좋아진다.

면접질문
HTML5 시멘틱 태그에 대해
=> header, nav, section, footer, aside..등은 div와 같은 동작을 한다.
=> 그러나 시멘틱 태그들은 의미를 가지고 있고 이에 따라 컨텐츠 구조를 더 정확히 파악할 수 있어서 SEO에 긍정적인 영향을 미칩니다.
