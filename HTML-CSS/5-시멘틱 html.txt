1. 시멘틱 HTML
=> 웹 페이지의 구조를 의미적으로 잘 표현할 수 있도록 html 요소를 사용하는 방법


2. 주요 시멘틱 HTML 요소
2-1. header(로고, 제목, 탐색 메뉴)
2-2. nav(내비게이션 링크)
2-3. main(페이지의 주요 컨텐츠)
2-4. section(문서 내의 구획을 정의)
2-5. article(독립적으로 배포 가능한 컨텐츠, 뉴스기사)
2-6. footer (문서의 푸터)
2-7. figure(삽화, 이미지, 다이어그램)


3. 시멘틱 HTML의 장점
3-1. SEO 개선
(검색 엔진 크롤러가 시멘트 HTML를 보고 각 컨텐츠가 무슨 의미를 가지는지 이해할 수 있고 페이지에 대한 효율적이고 정확한 평가가
가능하다.)
3-2. 유지보수 용이
3-3. 코드의 가독성 좋아진다.
