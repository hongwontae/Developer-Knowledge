바닐라 JS vs React 이벤트 처리 차이점

1. 핸들러 등록 방식
1-1. 바닐라 JS는 DOM 요소를 선택한 뒤, addEventListener를 통해 함수를 등록합니다. 또는 HTML 속성에 직접 접근하여 핸들러를
정의할 수 있습니다.
1-2. React는 JSX의 이벤트 속성에 접근하여 핸들러를 등록합니다.


2. 이벤트 객체
2-1. 바닐라 JS는 브라우저가 전달해주는 event 객체가 자동으로 전달됩니다.
2-2. React도 event 객체가 자동으로 전달되지만 Synthetic Event 시스템을 사용하여 이벤트를 최적화합니다.
=> 이 덕분에 React 가상 DOM과 통합됩니다.


3. 이벤트 바인딩과 바인딩 해제
3-1. 바닐라 JS는 이벤트를 등록하면 명시적으로 removeEventListener를 통해 해제해야 합니다.
3-2. React는 컴포넌트 생명주기에 의해서 핸들러가 자동으로 바인딩되거나 해제됩니다.
