1. React에서 State의 불변성은 어떻게 유지할 수 있나요?
=> 상태 객체를 직접적으로 변경하는 것은 React가 상태 변경을 감지하지 못하거나 성능 저하를 일으킬 수 있습니다.
=> 기존의 상태 객체의 불변성을 위해서 얕은 복사나 깊은 복사를 통해 기존 객체와 동일한 객체지만 주소 값이 다른 객체를
수정하여 사용하여야 합니다.


면접 질문
React에서 State의 불변성은 어떻게 유지할 수 있나요?
=> setState의 콜백 함수 인자 prevState를 깊은 복사하거나 얕은 복사하여 사용하여 return 하는 방식으로 불변성을
지킬 수 있습니다.
