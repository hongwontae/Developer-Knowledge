1. React Hooks
=> React 16.8에서 도입된 기능으로, 함수형 컴포넌트에서 상태와 생명주기 기능을 사용할 수 있게 해주는 함수들입니다.


2. 특징
2-1. 함수형 컴포넌트에서만 사용가능
2-2. 재사용성 => 커스텀 hooks를 생성해서 여러 컴포넌트에서 사용이 가능합니다.
2-3. 클래스의 복잡성 해결
2-4. hook는 컴포넌트 루트 레벨에서만 호출할 수 있습니다. 일반 함수, 조건문, 반복문에서 호출할 수 없습니다.


면접 질문
리액트 hooks에 대해서 설명해주세요
=> 리액트 16.8에서 도입된 기능으로, 함수형 컴포넌트에서 상태와 생명주기를 사용할 수 있도록 보장하는 함수들입니다.
=> 특징으로 함수형 컴포넌트에서만 사용가능, 커스텀 훅으로 만들어서 재사용 가능, 함수형 컴포넌트 root 레벨 안에서만
호출이 가능(조건문, 함수안, 반복문에서 호출 불가)가 있습니다.
