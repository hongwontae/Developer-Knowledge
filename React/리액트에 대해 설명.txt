1. 리액트란?
=> 사용자 인터페이스를 구축하기 위한 JS 라이브러리입니다. 주로 SPA 개발에 사용됩니다.

2. 주요 특징
2-1. 컴포넌트 기반
2-2. 가상 DOM
2-3. 단방향 데이터 흐름

3. 주요 개념
3-1. JSX
3-2. state
3-3. props
3-4. 라이프사이클 메서드

4. 장점
4-1. 재사용성
4-2. 빠른 렌더링
4-3. 풍부한 생태계

5. 단점
5-1. 학습 곡선
5-2. 자주 변하는 API
5-3. SEO 문제 : SSR, SSG를 통해서 해결 가능 => NextJS를 통해서 해결한다.
