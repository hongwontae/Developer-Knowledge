1. 기본 개념의 차이점
=> 클래스형 컴포넌트는 class로 정의하고 React.component를 상속받습니다. 생명주기 메서드를 통해 컴포넌트 생성부터
제거까지 관리할 수 있습니다.
=> 함수형 컴포넌트는 일반적인 함수로 정의하고 상태는 React Hooks를 통해 처리합니다. 생명주기 메서드가 존재하지 않지만
useEffect을 통해 유사한 흐름을 가질 수 있습니다.

2. 상태 관리
=> 클래스형 컴포넌트는 state를 직접 정의하고 setState 메서드를 통해 상태를 관리합니다.
=> 일반적으로 hooks를 사용하고 useState를 사용해서 상태를 관리합니다.


3. 문법과 가독성
=> 클래스형 컴포넌트는 this를 통한 상태와 메서드 접근, 생명주기 메서드의 사용으로 함수형 컴포넌트보다 문법이 복잡하고 가독성이
떨어집니다.
