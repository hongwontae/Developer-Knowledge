1. 자바스크립트의 메모리 관리에 대해 아는 대로 설명해주세요
=> JS 가비지 컬렉션을 통해 자동으로 메모리를 관리합니다.
=> 더 이상 어떤 변수나 객체에서도 접근할 수 없는 메모리는 GC가 자동으로 관리합니다.
=> 변수를 null로 변경하거나 함수 안에서 변수를 할당하고 해당 함수가 끝났을 떄를 의미합니다.


2. 콜 스택 (Call Stack)과 힙 (Heap)에 대해 설명해주세요.
=> JS 메모리는 콜 스택과 힙 메모리로 나뉩니다.
=> 콜 스택은 함수 실행 컨텍스트가 저장됩니다. 
=> 힙 메모리는 객체, 배열 같은 동적 데이터가 저장되는 곳으로, 크기가 정해져 있지 않고
동적으로 할당됩니다.


3. 즉시 실행 함수 (IIFE)에 대해 설명해주세요.
=> 정의되자마자 실행되는 함수를 의미합니다.
=> 전역 스코프를 오염시키지 않고 변수를 은닉할 떄 사용됩니다.
(내부 변수는 외부에서 접근할 수 없는 지역 변수로 유지되기 떄문입니다.)
=> (function(){})()/ (()=>{})()

4. 클래스에 대해 설명해주세요.
=> ES6에서 도입되었고 프로토타입 기반 객체 지향 프로그램밍을 쉽게 사용할 수 있게 합니다.
=> constructor로 객체를 초기화, extends를 통해 상속, super 키워드를 통해 부모 클래스 접근 가능합니다.


5. map과 forEach, reduce에 대해 설명해주세요.
=> 해당 메서드들은 배열을 다룰 떄 유용합니다.

5-1. map
=> 기존의 배열을 변환하여 새 배열을 만듭니다. 
5-2. forEach
=> 각 요소를 순회하며 콜백 실행하고 반환값이 없습니다.
5-3. reduce
=> 각 요소를 누적(accumulate)하여 하나의 값으로 반환하는 메서드입니다.
=> reduce(callback, initialValue)
=> callback의 인자는 (acc, cur, index, arr)=>{...}
