1. px, em, rem의 차이에 대해서 설명해주세요
1-1. px이란 고정된 단위로, 화면 크기와 관계없이 정해진 픽셀 크기를 사용합니다.
1-2. em이란 부모 요소의 글꼴 크기를 기준으로 상대적인 크기를 설정합니다.
1-3. rem이란 루트 요소(<html>)의 글꼴 크기를 기준으로 상대적인 크기를 설정합니다.


2. vw, vh에 대해 설명해주세요
2-1. vw는 뷰포트 너비의 1%을 의미합니다.
2-2. vh는 뷰포트 높이의 1%를 의미합니다.
=> 뷰포트란 브라우저에서 보이는 화면의 가시 영역입니다.


3. SCCS란 무엇인가요?
=> CSS의 전처리기인 Sass의 문법 중 하나로 변수, 중첩, 믹스인, 함수 등 코드 재사용과 구조화가 가능합니다.
=> 컴파일 시 일반 CSS로 변환됩니다.

3-1. SASS란?
=>  CSS의 전처리기로, CSS를 더 효율적이고 생산적으로 작성할 수 있게 도와주는 도구입니다.


4. 반응형 브레이크 포인트란? 반응형 브레이크 포인트를 어떻게 잡는지?
4-1. 화면 크기에 따라 디자인이 적용하도록 스타일을 변경하는 기준점을 의미합니다.
4-2. 주로 CSS의 미디어 쿼리나 Tailwind config.js에서 제어합니다.

4-3. 일반적인 브레이크 포인트는 320px, 768px, 1024px, 1440px으로 알고 있습니다.
=> 데스크탑 > 작은 노트북 > 테블릿 > 휴대폰
