1. dependencies 와 devDependencies 차이에 대해 설명해주세요.
=> 디펜덴시는 프로덕션 환경에 필요한 패키지입니다. (react, axios)
=> 데브디펜덴시는 개발 및 빌드 단계에서만 필요한 패키지입니다.


2. XSS와 CSRF에 대해 설명해주세요.
=> XXS(크로스 사이팅 스크립팅)은 공격자가 악위적인 스크립트를 삽입하여 사용자의 브라우저에서 실행되도록
유도하는 공격입니다.
(innerHtml 대신 textContent를 사용)
=> CSRF(크로스 사이팅 리퀘스트 Forgery)는 사용자가 의도하지 않은 요청을 보내도록 유도하는 공격입니다.
(CSRF 토큰, SameSite 쿠키..)


3. 인증과 인가에 대해 설명해주세요.
=> 인증은 사용자가 누군인지 확인하는 과정입니다.
=> 인가는 인증된 사용자가 특정 리소스에 접근할 권한이 있는지 확인하는 과정입니다.


4. package.json에서 private에 대해 설명해주세요.
=> private : true 설정 시 해당 패키지는 npm에 배포되지 않습니다.
=> 내부 프로젝트에서만 사용할 떄 설정합니답.


5. 프로젝트를 npm에 배포하려면 어떤 설정이 필요할까나요?
=> npm 배포는 내가 만든 라이브러리를 npm 레지스트리에 등록하여, 누구나 npm install로 설치할 수 있도록
하는 것을 의미합니다.
=> 프로젝트 생성 => npm 로그인 => npm publish --access public
