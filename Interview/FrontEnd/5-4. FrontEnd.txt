2. XSS와 CSRF에 대해 설명해주세요.
=> XXS(크로스 사이팅 스크립팅)은 공격자가 악위적인 스크립트를 삽입하여 사용자의 브라우저에서 실행되도록
유도하는 공격입니다.
=> 입력값 검증해야 합니다.

=> CSRF(크로스 사이팅 리퀘스트 Forgery)는 사용자가 의도하지 않은 요청을 보내도록 유도하는 공격입니다.
=> samesite(외부 사이트에서 보낸 요청에는 쿠키가 미포함), 출처 확인...


3. 인증과 인가에 대해 설명해주세요.
=> 인증은 사용자가 누군인지 확인하는 과정입니다.
=> 인가는 인증된 사용자가 특정 리소스에 접근할 권한이 있는지 확인하는 과정입니다.


6. 오버라이딩, 오버로딩
=> 오버라이딩 : 부모 클래스의 메서드를 자식 클래스에서 새로운 구현으로 덮어씁니다.
=> 오버로딩 : 동일한 이름을 가진 함수가 매개변수 개수나 타입에 따라 다르게 동작하기 만드는 것입니다.
