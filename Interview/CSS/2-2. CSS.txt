1. Padding과 Margin에 대해 알려주세요. 그리고 Margin 병합(Margin Collapse)이 무엇인가요?
1-1. padding이란 컨텐츠와 border 사이의 안쪽 여백입니다.
1-2. margin이란 border와 다른 요소 간의 바깥 여백입니다.

1-3. 마진 병합이란 수직 방향의 마진이 합쳐지면서 더 큰값만 적용되는 현상을 의미합니다.


2. Attribute와 Property에 대해서 알려주세요
2-1. Attribute(HTML)
=> HTML 요소에 정의된 속성입니다. 
=> 항상 문자열 형태로 사용되고, 초기값을 설정하는데 사용됩니다.

2-2. Property (JS)
=> DOM 객체의 속성으로, JavaScript로 접근하여 값을 읽거나 수정할 수 있습니다.
=> Attribute과 다르게 값의 타입이 다양합니다.


3. CSS 선택자 우선순위란 무엇인가요?
3-1. 여러 선택자가 동일한 요소에 적용될 떄, 어떤 스타일이 우선 적용될지를 결정하는 기준입니다.

3-2. inline > id > class, Attribute, 수도 class > 태그 선택자 > universal 선택자
=> 다만 !important 선택자는 최우선으로 적용됩니다.


4. id, class에 대해 설명해주세요
4-1. id는 HTML 문서 내에서 고유한 식별자 입니다.
=> css에서 #id로 선택할 수 있습니다.
=> 여러 태그에 하나의 id를 사용하면 브라우저 유효성 검사에서 오류가 발생합니다.

4-2. class는 공통 스타일을 적용할 수 있는 식별자 입니다.
=> css에서 .class로 선택할 수 있습니다.
