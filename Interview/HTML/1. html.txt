1. HTML이 무엇인가요?
=> 웹 페이지의 구조를 정의하는 *마크업 언어*입니다.
=> 다양한 컨텐츠를 브라우저가 렌더링할 수 있도록 구조화합니다.


2. HTML5이 무엇인가요?
=> HTML의 최신 표준으로, 시멘틱 태그, 멀티미디어 지원(audio, video), 
웹 애플리케이션 기능 강화(localstorage, geolocation, websocket..) 등 최신 웹 기술을 포함합니다.


3. 시멘틱 태그란 무엇인가요?
=> 콘텐츠의 의미를 명확히 전달하는 태그입니다.
=> 시멘틱 태그는 구조와 목적을 설명하여 검색 엔진 최적화(SEO)와 접근성을 향상시킵니다.
