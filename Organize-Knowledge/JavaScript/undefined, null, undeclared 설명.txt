1. undefined
=> 변수는 선언되었지만 값이 할당되지 않았을 떄 자동으로 가지는 값입니다.

2. null
=> 의도적으로 '값이 없음'을 표현하기 위해 할당하는 값입니다.

3.undeclared
=> 변수가 선언되지 않아 참조할 수 없는 상태입니다.

undefined, null, undeclared에 대해 비교하세요
=> undefined는 변수는 선언되었지만 값은 할당되지 않았을 떄 자동으로 가지는 값입니다.
=> null은 의도적으로 '값이 없음'을 표현하기 위한 값입니다.
