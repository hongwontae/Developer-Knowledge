1. 즉시 실행 함수
=> 함수가 정의되자마자 즉시 실행되는 패턴을 의미합니다.
=> 주로, 스코프를 캡슐화하고 변수의 범위를 제한하는데 사용합니다.


2. 형태 : (function(){})()
=> 함수 이름이 없는 함수 표현식을 중괄호로 덮습니다.
=> 그리고 호출합니다.


3. 사용목적
=> 해당 스코프를 독립적으로 유지한다.
=> 초기화 작업을 수행합니다.
