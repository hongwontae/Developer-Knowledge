1. 함수 선언문
=> function 키워드와 이름을 나열하고 중괄호와 대괄호를 사용해서 함수를 정의합니다.
=> 가장 큰 특징은 호이스팅으로 인해 선언 전에 호출해서 사용할 수 있습니다.


2. 함수 표현식
=> 함수를 변수에 할당해서 정의합니다.
=> 함수 표현식은 호이스팅이 적용되지 않아 선언 이후에 접근할 수 있습니다.


면접 질문
함수 선언문과 함수 표현식을 비교 설명하세요
=> 함수 선언문은 function 키워드, 함수이름, 매개변수를 받는 중괄호, 함수 본문 대괄호를 순서대로 작성하여 정의합니다
=> 가장 큰 특징은 호이스팅이 되어서 선언전에 호출될 수 있습니다.
=> 함수 표현식은 변수에 함수를 할당하여 정의합니다.
=> 함수 표현식은 호이스팅 적용되지 않아 선언전에 접근이 불가능합니다.
