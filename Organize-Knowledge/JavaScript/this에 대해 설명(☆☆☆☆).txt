1. this란 무엇인가?
=> 함수가 호출된 문맥에 따라 다르게 참조되는 객체입니다.
=> this는 함수가 호출되는 위치에 따라 동적으로 결정됩니다.


2. this 동작 방식
this는 크게 6가지 방식으로 다르게 참조합니다.
2-1. 전역 컨텍스트 : 브라우저는 this => window, nodeJS this => global 객체
2-2. 일반 함수 호출 : this => 전역 객체 참조(window)/ 엄격모드일 때는 undefined
2-3. 객체 메서드 : this => 메서드를 호출한 객체를 참조
2-4. 생성자 함수 : this => 새로 생성된 객체를 참조
2-5. 화살표 함수 : 상위 스코프의 this를 참조
2-6. bind, callm apply : bind의 경우 this를 명시적으로 설정하고 새로운 함수를 생성하고, apply, call의 경우 비슷하지만
함수를 바로 호출한다.
