1. Firewall
=> 외부의 불법적인 접근이나 해킹, 악성 트래픽으로부터 내부 네트워크를 보호하는 장치나 소프트웨어를 의미합니다


2. 주요 기능
2-1. 접근 제어
=> 허용된 IP, 포트, 프로토콜만 통과시키고, 의심스러운 연결은 차단합니다.

2-2. 트래픽 필터링
=> 패킷의 출발지/목적지 주소, 포트 번호, 프로토콜 등을 분석해 위험 요소를 걸러냅니다.

2-3. 애플리케이션 차단
=> 특정 프로그램(예: 토렌트, 게임)의 인터넷 접근을 제한할 수 있습니다.

2-4. 로그 기록 및 모니터링
=> 모든 차단/허용 이력을 기록해 보안 사고 발생 시 추적이 가능합니다.

2-5. NAT(네트워크 주소 변환)
=> 내부 네트워크의 IP를 외부에 노출하지 않도록 변환해 보안성을 높입니다.


3. 방화벽 종류
3-1. 네트워크 방화벽
하드웨어 기반(라우터, 전용 장비)으로, 기업이나 데이터 센터에서 사용됩니다.

3-2. 호스트 기반 방화벽
=> 개별 PC/서버에 설치되는 소프트웨어형 방화벽.

3-3. 차세대 방화벽(NGFW)
=> 기존 방화벽 + 침입 탐지(IDS), 애플리케이션 제어, 딥 패킷 검사 등 고급 기능 통합.
