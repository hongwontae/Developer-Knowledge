1. 메인보드란?
=> 컴퓨터의 모든 부품을 연결하는 중심 회로 기판입니다. (컴퓨터의 중앙 교차로)
=> 메인 보드의 전원 회로를 통해 PSU에서 온 전류를 CPU, RAM, 그래픽 카드에 공급합니다.
=> 메인 보드의 통신 회로는 CPU, RAM, 그래픽 카드 간의 빠르고 정확한 데이터 통신을 돕습니다.
=> 메인 보드는 USB 포트, HDMI 포트, PCle 슬롯 등을 제공합니다. 외부 장치나 추가 부품들을 
 연결할 수 있도록 해줍니다. (본체의 USB 포트는 메인보드의 USB 포트와 연결되어 있습니다.)


2. 그래픽 카드
=> 영상, 이미지, 게임 그래픽 등을 처리하는 전용 장치입니다.
=> CPU가 모든 그래픽 작업을 처리하면 너무 많은 연산이 필요하기에 
 그래픽 카드는 이를 전담해서 화면 출력 속도를 빠르게 해 줍니다.
=> 메인보드에 내장된 그래픽 칩셋이 있지만 고성능 작업을 위해 별도의 외장 그래픽 카드를 사용하는 경우가
 있습니다.
