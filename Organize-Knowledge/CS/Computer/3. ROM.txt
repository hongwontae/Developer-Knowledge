1. ROM (Read-Only-Memory)
=> 전원이 꺼져도 내용이 지워지지 않는 메모리입니다.
=> 일반적으로 ROM의 내용은 수정할 수 없고 일부 업데이트가 가능하도록 설계되어 있습니다.
=> 컴퓨터가 부팅될 떄 ROM의 코드 실행 => 저장된 펌웨어를 실행합니다.

1-1. ROM에는 펌웨어가 저장되어 있습니다.
1. BIOS(Basic Input/ Output System)
=> 컴퓨터가 부팅될 떄 가장 먼저 실행되는 기본 소프트웨어입니다.
=> CPU가 전원을 받으면 ROM에 저장된 BIOS or UEFI가 실행됩니다.
=> RAM, 저장장치 등 하드웨어를 초기화하고 CPU가 실행할 초기화 코드를 제공합니다.
=> 운영체제를 실행할 준비를 합니다.

=> 16비트 방식으로 동작합니다.

2. UEFI (Unified Extensible Firmware Interface)
=> BIOS를 대체하는 최신 소프트웨어입니다.
=> 그래픽 인터페이스 지원, 보안 기능 강화(Secure Boot), 빠른 부팅 속도 등의 장점이 있습니다.
=> 64비트 환경에서 동작하고, 최신 컴퓨터는 UEFI를 사용합니다.

3. Firmware라고 불리는 이유
=> 펌웨어란 하드웨어를 제어하고 동작을 관리하는 소프트웨어를 의미합니다.
=> 컴퓨터 같은 복잡한 시스템은 하드웨어가 제대로 동작하게 하는 저수준의 소트프웨어가 필요하기에
 펌웨어를 사용합니다.
