1. 임베딩이란?
=> 일반적으로 "내장" 또는 "삽입"을 의미합니다.


2. 데이터 임베딩
=> 데이터를 저차원 공간으로 변환하는 기술입니다. 
=> ex) 이미지를 백터 형태로 변환하는 이미지 임베딩

3. 웹 임베딩
=> Youtube, Google Maps 등의 컨텐츠를 다른 웹페이지에 임베드하여 보여주는 방식입니다.

4. 소프트웨어 임베딩
=> 소프트웨어나 애플리케이션의 일부로 다른 기능이나 서비스를 내장하는 것입니다.

5. 데이터베이스 임베딩
=> 애플리케이션 내에서 동작하는 경량화된 데이터베이스입니다. (Sqlite)

6. 임베디드 시스템
=> 특정 기능을 수행하는 특화된 컴퓨터 시스템입니다. 범용적인 작업이 아닌 특정 작업만을 위해 설계된 시스템입니다.
=> GPS, 화면 터치, Camera 등은 특정 기능을 위해 설계된 임베디드 시스템입니다.
