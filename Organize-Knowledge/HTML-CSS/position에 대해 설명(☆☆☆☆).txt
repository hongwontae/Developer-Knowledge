1. position
=> HTML 요소가 페이지 내에서 어떻게 배치될지 결정하는데 사용되는 속성입니다.

2. position의 종류
2-1. static : 기본값으로, 일반적인 문서 흐름을 따릅니다. top, bottom, left, right 속성이 적용되지 않습니다.
(인라인, 블록 요소)
2-2. relative : 요소를 기존의 위치에서 상대적으로 배치할 떄 사용됩니다.
2-3. absolute : 요소가 문서 흐름에서 제거되고, 가장 가까운 relative, absolute 부모 요소를 기준으로 배치됩니다.
2-4. fixed : 브라우저 창을 기준으로 배치되고, 스크롤해도 고정된 위치에 남아있습니다.
2-5. sticky : relative와 fixed의 조합으로 스크롤에 따라 특정 지점에 도달하면 요소가 고정됩니다.
