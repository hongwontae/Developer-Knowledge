1. JSX란 JavaScript Syntax Extension의 약어이고 리액트에서 UI를 정의하는데 사용됩니다.
2. JSX가 JS 객체로 변환되고 이를 통해 Virtual DOM을 생성하고 이후에 HTML로 생성됩니다.

3. 특징
3-1. 표현식 사용 : 대괄호를 사용해서 JS 표현식을 사용할 수 있습니다.
3-2. 조건부 렌더링 : 조건문이나 삼항 연산자를 사용해서 조건부 렌더링을 할 수 있습니다.
3-3. JSX의 태그들의 속성은 React 친화적이다. => key 같은 속성은 JSX에서 볼 수 있다.
