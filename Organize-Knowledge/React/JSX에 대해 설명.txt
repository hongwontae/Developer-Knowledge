1. JSX
=> 리액트에서 사용하는 확장 구문으로 JS 코드 내에서 HTML과 유사한 문법이 사용 가능한
JavaScript Syntax Extension이라고 부릅니다.
=> JSX를 통해서 컴포넌트의 구조를 정의하고, UI를 구성합니다.


2. 특징
2-1. div, p, section 같은 HTML 요소를 사용할 수 있습니다.
2-2. JS 표현식을 사용할 수 있습니다. => 중괄호 사용
2-3. 조건부 렌더링이 가능합니다.


면접 질문

=> 리액트에서 사용하는 확장구문이고 JS 코드 내에서 HTML처럼 사용할 수 있게 하는 JSyntaxExtention을 의미합니다.
=> 이를 통해서 컴포넌트를 구축하고 UI를 구축합니다.

=> 마지막으로 if, or, and 연산자, 삼항 연산자로 조건부 렌더링을 할 수 있습니다.
