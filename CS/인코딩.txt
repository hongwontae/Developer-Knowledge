1. 인코딩이란?
=> 데이터를 다른 형식으로 변환하는 과정을 의미한다.
=> 데이터를 효율적으로 저장하거나 전송하기 위해 사용된다.


2. 주요 인코딩 종류
2-1. 텍스트 인코딩 : 텍스트 데이터를 바이너리 데이터로 변환/ ASCII 인코딩 방식('A' : 65)
2-2. 이미지 인코딩 : 이미지 데이터를 특정 포맷으로 변환(JPEG, PNG, GIF)
2-3. 오디오 인코딩 : 아날로그 오디오 신호를 디지털 데이터로 변환(MP3, WAV)
2-4. 비디오 인코딩 : 비디오 데이터를 특정 포맷으로 변환(H.264, H.265..)


*** 대부분의 인코딩 방식은 효율적인 2 진법 데이터를 다루기 위한 방식이다.
