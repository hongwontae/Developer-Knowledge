1. path module
=> 파일과 디렉토리의 경로를 조작하고 처리하기 위한 모듈


2. path.join('./', 'fo', 'file.txt');
=> 여러 경로 조각을 결합하여 하나의 경로를 생성합니다. 경로 구분자는 현재 운영 체제에 맞게 자동으로 처리됩니다.


3. path.resolve();
=> 인자로 전달된 경로 조각을 결합하여 절대 경로를 반환합니다. 


4. path.basename()
=> 경로에서 파일 이름을 반환합니다. 선택적으로 파일 확장자를 제거할 수도 있습니다.


5. path.dirname()
=> 경로에서 디렉토리 부분을 반환합니다.


6. path.extname()
=> 경로에서 파일 확장자를 반환합니다.


7. path.isAbsolute()
=> 경로가 절대인지 boolean을 반환


8. __dirname
=> 현재 파일의 디렉토리 절대 경로를 반환한다.


9. __filename
=> 현재 파일의 디렉토리 + 파일 절대 경로를 반환한다.
